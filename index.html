AP CS A

CS A Summary

  1. Prerequisites: 10th Grade
  2. Target: For those who want to study CS after ORR
  3. Interest: Students need to want to write a lot of code
  4. Content: All programming
  5. Language: Java
  6. Test: Written exam and open response

AP CS Principles

CS Principles Summary

  1. Prerequisites: 10th Grade
  2. Target: Want to learn about fundamental CS topics
  3. Interest: Students need to be curious how computing impacts their lives and who want to learn programming
  4. Content: Mix of fundamental CS concepts & programming
  5. Language: Javascript
  6. Test: Written exam and independently written original program
